Feature List for eNcounterCloud

    General Features

Secure cloud-based medical image archiving and management. ​
FDA-accredited Picture Archiving Communications System (PACS). ​
HIPAA compliance for data security and privacy. ​
AES-256 encryption for data storage and communication. ​
TLS 1.2 encryption for data in transit. ​
Lossless PNG compression for individual images. ​
H264 MP4 compression for multi-frame DICOM formats (video). ​
Compatibility with DICOM/TLS services and VPN tunnels. ​
Not intended for mammography or disease diagnosis, treatment, or prevention. ​

    System Requirements

Minimum hardware requirements: 1024x768 resolution, Intel Core Duo, 2GB RAM. ​
Supported browsers: Internet Explorer, Firefox, Google Chrome, Safari, Microsoft Edge. ​
Minimum bandwidth requirement: 1.5 Mbps. ​

    User Management

1 Creation of user accounts via the GlobalMed Account Management Portal. ​
2 Support for seven user profiles (clinical and administrative users).
3 Clinical user functions: viewing studies, managing patients, conducting consultations, transferring studies, uploading reports, sharing studies.
4 Administrative user functions: assigning patients to physicians, establishing routing rules, managing worklists, and PHI logs. ​
5 Traceability of clinician-reviewed studies through unique accounts. ​

    Login and Registration ​

1 Initial admin username creation by GlobalMed. ​
2 Secure password creation during registration. ​
3 "Forgot Password" feature for account recovery. ​
4 Passcode-based access to specific studies. ​

    Account Management

1 Modify account details: email, SMS number, specialties, address, phone, etc. ​
2 Enable/disable email notifications for study updates. ​
3 Change password with optional password hint.

    Clinical User Features ​

1 View and manage pending studies. ​
2 View and manage reviewed studies. ​
3 Share studies within institutions or via passcodes. ​
4 Search for studies using filters (date, time, custom fields). ​
5 Study Viewer functionalities:

    Image stack navigation. ​
1 Video player for multi-frame images. ​
2 Image adjustments (zoom, contrast, brightness, inversion, etc. ​).
3 Report creation with templates (HPI, SOAP, radiologist reports, etc. ​).
4 Notes creation for case-specific details. ​
5 Study sharing with institutions or specific recipients. ​
6 DICOM header viewing. ​
7 Mark studies as reviewed. ​
8 Attach PDFs to studies (up to 5 MB, no duplicates). ​

    Administrative User Features ​

1 Patient administration: add, edit, delete patient profiles. ​
2 Assign physicians to patients. ​
3 Transfer patients between institutions. ​
4 Schedule studies for patients. ​
5 Manage patient studies: generate passcodes, transfer studies, delete studies, view/delete series/images. ​
6 Manage worklists: search, modify, delete worklist items. ​
7 Routing management: add, edit, delete routing rules for DICOM nodes. ​
8 PHI logs: download access reports in .csv format with detailed logging information. ​

This list isolates each feature for independent testing purposes.
